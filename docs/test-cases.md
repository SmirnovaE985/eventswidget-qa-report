# Тест-кейсы

## Критичные сценарии

| ID | Название тест-кейса | Предусловия | Шаги | Ожидаемый результат |
|---|---|---|---|---|
| TC-01 | Открытие страницы конструктора | Браузер открыт | 1. Перейти по ссылке https://dev.3s.info/eventswidget/ | Страница успешно загружается, основные блоки интерфейса отображаются корректно |
| TC-02 | Генерация превью с валидными параметрами | Открыта страница конструктора | 1. Выбрать тематику 2. При необходимости выбрать страну 3. Указать размеры блока 4. Выбрать цветовую тему 5. Нажать «Сгенерировать превью» | Превью успешно отображается без ошибок |
| TC-03 | Генерация iframe-кода после выбора параметров | Открыта страница конструктора | 1. Указать параметры виджета 2. Нажать «Сгенерировать превью» 3. Прокрутить страницу до блока со сгенерированным кодом | На странице отображается iframe-код для вставки |
| TC-04 | Проверка передачи параметра `theme` для выбранной темы | Открыта страница конструктора | 1. Выбрать любую цветовую тему 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде присутствует параметр `theme`, соответствующий выбранной теме |
| TC-05 | Проверка передачи параметра `theme` для красной темы | Открыта страница конструктора | 1. Выбрать красную тему 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде присутствует параметр `theme` с корректным значением для красной темы |
| TC-06 | Проверка передачи параметра `event_type` при выборе одной тематики | Открыта страница конструктора | 1. Выбрать одну тематику 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде присутствует параметр `event_type`, соответствующий выбранной тематике |
| TC-07 | Проверка передачи параметра `event_type` при выборе нескольких тематик | Открыта страница конструктора | 1. Выбрать несколько тематик 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде параметр `event_type` содержит все выбранные значения |
| TC-08 | Проверка изменения ширины превью при вводе значения в поле «Ширина, px» | Открыта страница конструктора | 1. Ввести новое значение в поле «Ширина, px» 2. Нажать «Сгенерировать превью» 3. Сравнить ширину превью | Ширина превью изменяется в соответствии с введённым значением |
| TC-09 | Проверка изменения высоты превью при вводе значения в поле «Высота, px» | Открыта страница конструктора | 1. Ввести новое значение в поле «Высота, px» 2. Нажать «Сгенерировать превью» 3. Сравнить высоту превью | Высота превью изменяется в соответствии с введённым значением |
| TC-10 | Проверка режима «на всю ширину контейнера» | Открыта страница конструктора | 1. Активировать опцию «на всю ширину контейнера» 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде ширина указывается как `width="100%"` |
| TC-11 | Проверка режима «на всю высоту блока» | Открыта страница конструктора | 1. Активировать опцию «на всю высоту блока» 2. Нажать «Сгенерировать превью» 3. Проверить iframe-код | В iframe-коде высота указывается как `height="100%"` |
| TC-12 | Проверка визуального соответствия превью режиму `width=100%` и `height=100%` | Открыта страница конструктора | 1. Активировать «на всю ширину контейнера» 2. Активировать «на всю высоту блока» 3. Нажать «Сгенерировать превью» 4. Оценить отображение превью | Превью визуально соответствует выбранным настройкам размеров либо поведение явно объясняется интерфейсом |
| TC-13 | Проверка соответствия выбранной темы и визуального отображения превью | Открыта страница конструктора | 1. Выбрать светлую тему 2. Нажать «Сгенерировать превью» 3. Сравнить ожидаемую и фактическую цветовую гамму | Превью отображается в цветовой гамме, соответствующей выбранной теме |
| TC-14 | Проверка читаемости заголовков таблицы в светлых темах | Открыта страница конструктора | 1. Выбрать синюю или зелёную светлую тему 2. Нажать «Сгенерировать превью» 3. Проверить заголовки таблицы | Текст заголовков таблицы читаем и имеет достаточный контраст относительно фона |

## Примечание
Тест-кейсы ориентированы на проверку критичных пользовательских сценариев и основной бизнес-логики конструктора.
| TC-15 | Проверка копирования сгенерированного кода | Сгенерирован iframe-код | 1. Нажать «Скопировать код» 2. Вставить содержимое в текстовый редактор | В буфер обмена копируется полный и корректный iframe-код |
