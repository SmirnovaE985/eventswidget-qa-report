# Чек-лист

## 1. Базовая функциональность
- [ ] Страница открывается по URL `https://dev.3s.info/eventswidget/`
- [ ] Основные блоки страницы отображаются корректно
- [ ] Выбор тематики доступен
- [ ] Выбор страны доступен
- [ ] Поле `Ширина, px` доступно для ввода
- [ ] Поле `Высота, px` доступно для ввода
- [ ] Выбор цветовой темы доступен
- [ ] Кнопка `Сгенерировать превью` доступна
- [ ] После генерации отображается превью
- [ ] После генерации отображается iframe-код

## 2. Проверка generated code
- [ ] В сгенерированном коде присутствует `iframe`
- [ ] В `iframe` передаётся параметр `event_lang`
- [ ] В `iframe` передаётся параметр `event_type` при выборе тематики
- [ ] В `iframe` передаётся параметр `theme` при выборе темы
- [ ] При выборе нескольких тематик в коде передаются все выбранные значения `event_type`
- [ ] При выборе режима `на всю ширину контейнера` в коде выставляется `width="100%"`
- [ ] При выборе режима `на всю высоту блока` в коде выставляется `height="100%"`
- [ ] Код обновляется после изменения настроек
- [ ] Сгенерированный код доступен для копирования

## 3. Обязательная проверка темы «Красная»
- [ ] Выбрана тема `Красная`
- [ ] Сгенерировано превью для красной темы
- [ ] Проверено значение параметра `theme` в generated code
- [ ] Проверено соответствие выбранной темы и отображения превью

## 4. Проверка размеров блока
- [ ] Проверено изменение ширины через поле `Ширина, px`
- [ ] Проверено изменение высоты через поле `Высота, px`
- [ ] Проверен режим `на всю ширину контейнера`
- [ ] Проверен режим `на всю высоту блока`
- [ ] Проверена комбинация `width=100%` и `height=100%`
- [ ] Проверено соответствие preview и настроек размеров

## 5. UI/UX
- [ ] Проверена читаемость текста в превью
- [ ] Проверена читаемость заголовков таблицы в светлых темах
- [ ] Проверено визуальное соответствие светлой темы и превью
- [ ] Проверено визуальное соответствие тёмной темы и превью
- [ ] Проверено, что выбранная тема визуально различима
- [ ] Проверено, что превью не вводит пользователя в заблуждение относительно выбранных параметров

## 6. Адаптивность / responsive behavior
- [ ] Проверено поведение превью при фиксированных размерах
- [ ] Проверено поведение превью при относительных размерах (`100%`)
- [ ] Проверено, что превью не выходит за пределы контейнера
- [ ] Проверено наличие/отсутствие растяжения превью на весь контейнер
- [ ] Проверено соответствие generated code и визуального результата

## 7. Network / Console
- [ ] Открыта вкладка Network в DevTools
- [ ] Проверено наличие сетевой активности при генерации превью
- [ ] Зафиксировано преобладание аналитических запросов в Network
- [ ] Проверено отсутствие/наличие явного бизнес-API-запроса
- [ ] Проверена передача параметров через generated code
- [ ] Проверено отсутствие критических ошибок в Console

## 8. Edge cases
- [ ] Проверена генерация превью после выбора одной тематики
- [ ] Проверена генерация превью после выбора нескольких тематик
- [ ] Проверена смена нескольких тем подряд
- [ ] Проверена генерация после смены размеров
- [ ] Проверена генерация после включения режимов `100%`
- [ ] Проверено поведение полей размеров при переключении режимов

## 9. Копирование кода
- [ ] Кнопка `Скопировать код` доступна
- [ ] Код копируется в буфер обмена
- [ ] После копирования вставляется корректный iframe-код

## Примечание
Чек-лист отражает основные и критичные проверки, выполненные в рамках исследовательского тестирования страницы конструктора виджета.
